如何部署hexo博客到云服务器

发表于 2017-07-05 | 分类于 blog | 阅读次数:

晤，这篇文章主要介绍一下对于已有的hexoblog如何部署到自己的服务器。

部署本地SSH公钥文件到服务器（已部署的忽略）

首先homebrew安装ssh-copy-id

brew install ssh-copy-id

执行以下命令

ssh-copy-id user@host

其中将user替换为自己服务器用户名，host替换为对应的ip地址。通过此命令可以将本地的ssh公钥发送到目标主机上，然后登陆主机账户即可免密码登陆。

校验是否配置成功

ssh user@host echo "test"

配置hexo部署命令

在package.js中的dependencies下加入以下代码

"hexo-deployer-rsync": "git+https://github.com/dolonfly/hexo-deployer-rsync.git"

执行

npm install

配置config.yml

deploy:
  type: rsync
  host: your host
  user: user
  root: /usr/local/nginx/html/blog/
  port: 22
  delete: false

完成，但是这个单独发布到服务器，。回头研究一下如何同步发到github。
