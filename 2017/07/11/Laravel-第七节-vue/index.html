Laravel-第七节-vue

这节主要研究下laravel如何结合vue使用，虽然不是很喜欢这个方式=-=。

配置

目前使用的是laravel5.4版本

1.在laravel项目中的package.json中的devDependencies中进行如下配置

"vue": "^2.3.4",
"vue-resource": "^1.3.4",
"vue-router": "^2.7.0"

2.配置完了当然是安装依赖

npm install

也可使用npm install {package_name} --save-dev安装所需

3.执行

npm run dev

不同于独立版，执行npm run dev时Webpack会执行webpack.mix.js，生成public/css/app.css和public/js/app.js。

编写Vue组件

resources/assets/js/components/Example.vue是示例，可以作为参考。

在welcome.blade.php中进行如下调用

<div id="app">
    <example></example>
</div>
<script src="{{ asset('js/app.js') }}"></script>

如果Uncaught TypeError: Cannot read property 'csrfToken' of undefined

<script>
    window.Laravel =
    @php echo json_encode([
        'csrfToken' => csrf_token(),
    ]);
    @endphp
</script>

POST请求

Ajax需要带上CSRF，模板中设置如下。

<meta name="X-CSRF-TOKEN" content="{{csrf_token()}}">

请求中需要在headers里带上这个Token，不然的话会报500错误。

this.$http.post('http://localhost:8080/user/loginIn',
    {JSON:JSON.stringify(this.LoginInfo)},
    {headers: {'X-CSRF-TOKEN': $('meta[name="X-CSRF-TOKEN"]').attr('content')}, 
    emulateJSON: true})
    .then(function(response) {
        console.log(response);
    }, 
    function(response) 
    {
        console.log("===========this is error==========");
        console.log(response);
        return response;
    });
