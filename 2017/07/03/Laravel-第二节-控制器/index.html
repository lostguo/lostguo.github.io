Laravel-第二节-控制器

发表于 2017-07-03 | 分类于 laravel | 阅读次数

这一节主要记录一下在laravel中controller层的创建使用。

新建控制器

在项目app/Http/Controllers中新建一个Api/InterfaceController.php

namespace App\Http\Controllers\Api;

use Illuminate\Http\Request;
use App\Http\Controllers\Controller;

class InterfaceController extends Controller
{

    public function index()
    {
        return 'success! interface\'s index function had bind route';
    }


    // 带参数   俩种方式都可以获得参数
    public function selectInfo(Request $request,$id){
            echo $id;
            return 'select currentID:'.$request->id;
    }

}

中间件待补充

控制器和路由关联

在route/web.php需要进行绑定，如下：

Route::get('interface/index', 'Api\InterfaceController@index');


Route::get('interface/select/{id}', 'Api\InterfaceController@selectInfo');

如此就成功的建立了一个控制器并和路由进行绑定，并成功进行访问。
