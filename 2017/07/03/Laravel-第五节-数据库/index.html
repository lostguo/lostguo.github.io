Laravel-第五节-数据库

发表于 2017-07-03 | 分类于 laravel | 阅读次数

这一节主要记录下在Laravel中如何操作数据库。

数据库连接

在/config/database.php可以发现'host' => env('DB_HOST', '127.0.0.1')之类的代码。
env代表的是我们需要在.env文件中

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=c_api
DB_USERNAME=root
DB_PASSWORD=123

如果需要读写分离呢？

'mysql' => [
    'read' => [
        'host' => '192.168.1.1',
    ],
    'write' => [
        'host' => '196.168.1.2'
    ],
    'driver'    => 'mysql',
    'database'  => 'database',
    'username'  => 'root',
    'password'  => '',
    'charset'   => 'utf8',
    'collation' => 'utf8_unicode_ci',
    'prefix'    => '',
],

也许你需要

php artisan cache:clear 
php artisan config:clear

使用

查询

public function showApi(){
    $users = DB::select('select * from c_user where userID = ?', [1]);
    dd($user);
    return '';
}

新增

DB::insert('insert ignore into c_user (account, password) values (?, ?)', ['cxj', '123']);

返回值为Bool。

修改

$num = DB::update('update c_user set password = ? where account = ?', ['1990','cxj']);

返回受影响的行。

删除

delete 方法用于删除已经存在于数据库的记录。

$num = DB::delete('delete from c_user');

如同 update 一样，删除的行数将会被返回。

自增

$num = DB::table('c_user')->increment('age');
$num = DB::table('c_user')->increment('age',3);
$num = DB::table('c_user')->decrement('age');
$num = DB::table('c_user')->decrement('age',3);

其他

监听

在app/Providers/AppServiceProvider.php中进行监听设置

public function boot()
{
    DB::listen(function ($query) {
       // $query->sql
       // $query->bindings
       // $query->time
    });
}

事务

DB::transaction(function () {
    DB::table('users')->update(['votes' => 1]);

    DB::table('posts')->delete();
});

手动操作事务

如果你想要手动开始一个事务的回滚和提交操作，你可以使用 DB facade 的 beginTransaction 方法。

DB::beginTransaction();

你也可以通过 rollBack 方法来回滚事务：

DB::rollBack();

最后，可以通过 commit 方法来提交这个事务：

DB::commit();
