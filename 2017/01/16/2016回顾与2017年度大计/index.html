2016回顾与2017年度大计

发表于 2017-01-16 | 分类于 life | 阅读次数

晤，2016还是需要总结一下的，2017还是需要展望一下的，挖掘出自己内心最真实的渴望，然后，去拼命吧。

先理理自己的人生蓝图吧，看看自己还有多远。

生活：

有一个温柔美丽的媳妇，有一个萌萌可爱的闺女，有一个调皮的儿子，和谐的一家子能一年四季有时间在每个美丽的地方享受美好的生活，我搂着媳妇，我的闺女弹着钢琴，我的儿子静静看书，幸福呀，美滴很。（媳妇条件已经达成哈哈）

工作：

对游戏一直有一种梦，一直希望自己以后能创立一家暴雪一样的公司，希望自己能做出工作学习锻炼与游戏结合在一起的游戏，让自己的游戏成为一种经典，感觉必须通过虚拟现实，虚拟现实一定也能通过活动产出不低于现实生活的价值，感觉虚拟现实是建设精神体系的基础核心。

当我怀揣着这些梦想的时候，我的2016我在干嘛呢？

我在公司写app接口，作为一枚php工程师，我也是web型全栈发展的。

收获：

  1. 前端技术得到一个较大的提升，bootstrap相对熟练了，前端框架vue系列得到了研究，比较欣赏这种webapp型的。
  2. php研究了tp5，和感觉已经忘了的symfony+laravel5，因为用的并不算太深入，但是明了了一部分路由的思想。
  3. linux使用的比以前更溜点，因苹果废弃pptp服务搭建了L2TP的vpn，废弃掉原有博客，搭建hexoblog，简单用了用nodejs、npm，开始研究起ios。
  4. 看了《如何阅读一本书》《心理控制术》《毛泽东传》(一大半)《意志力：关于专注、自控与效率的心理学》对于自己的好学精神起到促进作用。
  5. 搭了个小说看书网页，从其他网站抓取数据，从此自己不用纠结扰人的广告。
  6. 跑步大计跑了断，断了跑，直到搬到和媳妇一起再没跑，但是又发现了，不要小瞧任何一点运动，每天早晚各50个深蹲，目前保持的不错，允许自己每周遗漏个2次，作为身体休息日，目前加入了标准俯卧撑，改跑步为室内运动，成就感相对小，但是感受到的最大的好处就是，陪媳妇逛街，我媳妇走不动了，而我还好，这是最大的进步，继续保持。

夭折：

  1. 折腾了几天unity3D，折腾了1个月C++，我可怜的课程费，因为时间短，并没有显著的收获，以及目标还不够切实，有些模糊，有些自己的想当然看法遮挡。
  2. 可怜的考试。
  3. 每日看书30min的计划没有保持的太好，每周大概只能保持4天，一个是因为一本书看完不知道下一本看什么书，需要解决书单来源问题，二是会受手游的影响，已经认识到此项问题，不思议迷宫很好玩，但是因为它我也产生了将手游也将作为PC游戏一个级别去处理，因为它少了很多碎片时间，醒悟的不算太晚。

2016年总体来说还是不错，学习能力比较显著的提升了，游戏已经玩的少了，因为驱动我前进的目标变成我自己去做一个游戏用这个游戏讲故事。

2017年度计划：

技术：

  1. 《ios开发指南 从Hello World到App Store上架》研究好
  2. 设计一个成就系统app（普通成就，隐藏成就）
  3. 设计一款游戏（收集类，惊悚类）
  4. 细化各类目标并制定目标方案
  5. 深入理解c++
  6. php提升计划

生活：

  1. 感兴趣的东西太多，如何融合起来。
  2. 深入挖掘爱好 唱歌、跳舞
  3. 陪伴宝宝
  4. 学好英语
  5. 保持锻炼习惯
  6. 每天看书1小时一定保持住(限：技术书，各类人文工具书，不包含畅销书与小说)
  7. 做事更专注
