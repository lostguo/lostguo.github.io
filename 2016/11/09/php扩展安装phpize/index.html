phpize安装php扩展，GD示范

今天正好要用到GD库的时候发现服务器没有开启GD，服务器也没有这服务，然后就以此文来演示一下安装过程。

依赖

首先作者一如既往的使用centos7，安装gd函数库也是需要需要一些依赖的，如下

shell> yum install -y libjpeg-devel libpng-devel freetype-devel


安装

依赖无误安装后，进入到你php的安装文件下的ext/gd目录

shell> cd /php-7.0.9/ext/gd

在目录下执行phpize，如果不知道phpize路径，可以通过命令find / -name phpize来查看你自己的phpize路径

shell> /usr/local/php7/bin/phpize

结果类似如下

Configuring for:
PHP Api Version:         20151012
Zend Module Api No:      20151012
Zend Extension Api No:   320151012

然后需要把它配置到php，不知道php-config路径的朋友同样可以通过命令find / -name php-config来查看

shell> ./configure --with-php-config=/usr/local/php7/bin/php-config
shell> make

(ps:使用发现未支持libjpeg./configure --with-php-config=/usr/local/php7/bin/php-config --with-jpeg-dir=/usr/local/jpeg/)

make完毕无误后的朋友会在当前目录的moudles下生成一个gd.so,将他拷贝到你php的扩展目录下

不知道自己php扩展目录的朋友，可以自行设置或者使用phpinfo查看extension_dir，抑或vi查看php.ini查看extension_dir

shell> cp modules/gd.so /usr/local/php7/lib/php/extensions/no-debug-non-zts-20151012/

复制到扩展目录后，在php.ini中添加如下内容

extension=gd.so

完毕，php的其它扩展也可以参照此文。
