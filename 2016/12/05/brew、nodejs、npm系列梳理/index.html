homebrew、nodejs、npm系列梳理

恩，linux和mac的工具问题一直不熟练与用完忘，今天梳理下mac下nodejs、homebrew。

brew

参考地址：homebrew官网地址

当更新时:需要设置/usr/local 目录切换为非root拥有者,不然brew update 要么没写入权限，要么root权限过高，升级完了，再把/usr/local权限切回去

brew doctor

安全检查，确保安全可靠。 针对性逐条处理。

中科大的源，据说比较好，不然brew慢死了= =

cd "$(brew --repo)"
git remote set-url origin git://mirrors.ustc.edu.cn/brew.git


node

之前是按照官网安装的node 但是没有用brew，所以用homebrew重装一遍了，删的乱七八糟的赶脚，但是这样子后面好管理。


乱入PHP

brew install homebrew/php/php70  --with-homebrew-openssl --with-homebrew-libxslt --with-homebrew-curl --with-fpm --with-debug --with-gmp --with-imap --with-tidy --with-libmysql


关于VUE

vue的需要用npm安装一些列其他工具，和hexoblog类似，正式发布的时候只需要发布需要的文件到服务器上。和安卓的思路一样么？
